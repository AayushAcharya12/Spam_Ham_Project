# Spam vs Ham Classifier

A Machine Learning project that detects whether a message is Spam or Ham (Not Spam) using NLP and TF-IDF features.

##  Live Features

* Text preprocessing (NLTK)
* TF-IDF Vectorization
* Logistic Regression model
* Streamlit Web App

##  Tech Stack

* Python
* Scikit-learn
* NLTK
* Streamlit

##  Model Performance

* Accuracy: ~97% and ~95%
* Confusion Matrix included

##  Run Locally

pip install -r requirements.txt
streamlit run app.py
